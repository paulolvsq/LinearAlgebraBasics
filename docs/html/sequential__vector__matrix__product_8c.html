Computes the product of a matrix A and a vector X sequentially.

This function calculates the vector result = A * X, where A is a matrix and X is a vector. The computation is performed sequentially without parallelization.

Parameters
APointer to the input matrix (size: A_rows x A_columns).
A_rowsNumber of rows in the matrix A (must be positive).
A_columnsNumber of columns in the matrix A (must equal dimension).
XPointer to the input vector (size: dimension).
dimensionDimension of the vector X (must equal A_columns).
Returns
Pointer to the resulting vector (size: A_rows) on success, or NULL on failure due to invalid dimensions, null pointers, or memory allocation errors.

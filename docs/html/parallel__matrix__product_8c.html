Computes the product of two matrices P and Q in parallel using OpenMP.

This function calculates the matrix product C = P * Q using parallelization to improve performance. Each element of the resulting matrix is computed by summing the products of corresponding elements from rows of P and columns of Q.

Parameters
PPointer to the first input matrix (size: P_rows x P_columns).
P_rowsNumber of rows in matrix P (must be positive).
P_columnsNumber of columns in matrix P (must equal Q_rows).
QPointer to the second input matrix (size: Q_rows x Q_columns).
Q_rowsNumber of rows in matrix Q (must equal P_columns).
Q_columnsNumber of columns in matrix Q (must be positive).
Returns
Pointer to the resulting matrix (size: P_rows x Q_columns) on success, or NULL on failure due to invalid dimensions, null pointers, or memory allocation errors.
